## Installation

**Install my-project with npm**

```bash
  npm create vite@latest
  cd project_name
  npm install

```

**To install Tailwind** 

```bash
   npm install -D tailwindcss postcss autoprefixer
   npx tailwindcss init -p
```

**To Run**
```bash
   npm run dev
```
